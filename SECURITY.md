# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.**

Instead, use [GitHub's private vulnerability reporting](https://github.com/b0x42/paperless2papra/security/advisories/new) to submit your report.

You should receive a response within 48 hours. We'll work with you to understand the issue and coordinate a fix before any public disclosure.
